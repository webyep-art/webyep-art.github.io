Luna Programming Language

WORK IN PROGRESS - Contains bugs and unfinished features.

Overview

Luna is a static-typed scripting language designed for simplicity and ease of use. It supports basic programming constructs, functions, arrays, and memory management.

Key Features

+
  • Basic data types (int, float, bool, string)
  • Arrays (int[], float[], bool[], string[])
  • Pointers and memory management
  • Functions
  • Control flow (if/else, while loops)
  • Standard library functions
  • Error handling with stack traces

Language Features

Data Types

+

Primitive Types:

int x = 42
float pi = 3.14
bool flag = true
string greeting = "Hello, Luna!"

Composite Types:

int[] numbers = [1, 2, 3]
ptr<int> p = alloc(int, 10)

Control Structures

+

If/Else:

if x > 10 {
  // do something
} else {
  // do something else
}

While Loop:

while x < 100 {
  x = x + 1
}

Functions

+
func add(int a, int b) -> int {
  return a + b
}

// Function call
int result = add(5, 7)

Memory Management

+
// Allocation
ptr<int> p = alloc(int, 10)

// Deallocation
free(p)

Philosophy

Code Clarity

+

A good programmer only writes code they understand. While comments in code are valuable for documentation, they are not a substitute for writing clear, self-explanatory code.

Luna's philosophy emphasizes writing code that is inherently understandable rather than relying heavily on comments.
