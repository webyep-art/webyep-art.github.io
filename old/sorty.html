Sorty

Lightweight File Organizer - Keep your downloads folder tidy with this system tray utility.

Overview

A minimalistic application that lives in your system tray and automatically organizes files in your downloads folder with a single click.

View on GitHub

Features

Smart Sorting

Automatically categorizes files into folders: Images, Audio, Archives, Torrents, and more.

System Tray Integration

Runs quietly in the background with a simple right-click menu.

Customizable

Change the target directory or add your own file categories.

Notifications

Get feedback after each sorting operation.

How It Works

Simple Setup

+
  1. Download and install the application
  2. It will appear in your system tray
  3. Right-click to access the menu
  4. Click "Sort!" to organize your files

Supported File Types

+

Images

JPG, PNG, GIF, WEBP

Audio

MP3, WAV, FLAC

Archives

ZIP, RAR, 7Z

Documents

PDF, DOCX, XLSX
