Pygame 3D Renderer

Pure Python 3D Rendering - No OpenGL, just Pygame and math.

Overview

A lightweight 3D rendering engine built entirely with Pygame, implementing core 3D graphics concepts from scratch without relying on OpenGL or other 3D libraries.

Quick Start

Basic Example

+
import pygame
from corelibs import core
from corelibs.transformations import Vector3

# Initialization
pygame.init()
screen = pygame.display.set_mode((800, 600))
camera = core.Camera(Vector3(0, 0, -5), 60)

# Create a 3D triangle
triangle = core.Triangle(
  Vector3(-1, -1, 0),
  Vector3(1, -1, 0),
  Vector3(0, 1, 0)
)

# Main loop
while True:
  # Rotate and render
  triangle.rotate(Vector3(0, 0.01, 0))
  triangle.draw(screen, camera)
  pygame.display.flip()

Technical Details

Core Components

+

The renderer implements:

  • 3D to 2D projection with perspective correction
  • Vertex transformations (rotation, translation)
  • Backface culling
  • Basic shading based on face normals
  • Render queue for proper depth sorting

Performance

+

While not as fast as hardware-accelerated solutions, the renderer is optimized for Python:

  • Matrix operations are implemented efficiently
  • Only necessary calculations are performed
  • Render queue minimizes overdraw
  • Perfect for educational purposes and simple 3D visualizations
